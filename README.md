## ğŸ› Troubleshooting

### Common Issues

**âŒ Metadata fetch fails**
- Verify URL is correct and public
- Update yt-dlp: `pip install -U yt-dlp`
- Check internet connection

**âŒ Download fails**
- Check FFmpeg is available
- Verify disk space
- Try different video

**âŒ QR code downloads deleted on mobile**
- This happens on HTTP (localhost)
- Deploy with HTTPS for mobile downloads
- Or use ngrok for local HTTPS testing

**âŒ Glare effect not showing**
- Clear browser cache (Ctrl+Shift+R)
- Check browser console for errors
- Verify `glare-hover.js` is loaded

## ğŸ“ API Reference

### Endpoints

#### `POST /api/metadata`
Fetch video or playlist information

**Request:**
```json
{
  "url": "https://www.youtube.com/watch?v=..."
}
```

**Response:**
```json
{
  "success": true,
  "kind": "video",
  "video": {
    "title": "Video Title",
    "duration": 180,
    "thumbnail": "https://...",
    "formats": [...]
  }
}
```

#### `POST /api/download`
Start download job

**Request:**
```json
{
  "url": "https://www.youtube.com/watch?v=...",
  "kind": "mp4",
  "format_id": "best"
}
```

**Response:** File download with `X-Download-Id` header

#### `GET /files/{download_id}`
Retrieve downloaded file (for QR codes)
